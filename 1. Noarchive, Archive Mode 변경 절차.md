순서
1. startup nomount상태에서 spfile parameter 변경
2. pfile을 spfile과 동기화
3. Database mount 상태로 변경하고 archive log mode 확인
4. Database log mode를 archive mode로 변경
5. Database open 상태로 변경하고 system switch logfile 후 archiving 파일 확인

## 1. startup nomount상태에서 spfile parameter 변경
  - log_archive_dest_1 : 아카이브 파일을 저장할 경로 지정
  - log_archive_format : 아카이브 파일의 이름 형식 지정

```bash
SYS @ orcl > startup nomount

SYS @ orcl > alter system set log_archive_dest_1='LOCATION=/home/oracle/arch';

System altered

SYS @ orcl > alter system set log_archive_format='ORCLDB%t_%s_%r.arc';
alter system set log_archive_format='ORCLDB%t_%s_%r.arc'
                 *
ERROR at line 1:
ORA-02095: specified initialization parameter cannot be modified

#log_archive_format 파라미터는 동적으로 변경될 수 없다

SYS @ orcl > alter system set log_archive_format='ORCLDB_%t_%s_%r.arc' scope=spfile;

System altered.

#%t : 인스턴스 번호, %s : 로그 시퀀스 번호, %r : 데이터베이스 식별 번호

SYS @ orcl > shutdown immediate
SYS @ orcl > startup

SYS @ orcl > show parameter archive

log_archive_dest_1                   string      LOCATION=/home/oracle/arch
log_archive_format                   string      ORCLDB_%t_%s_%r.arc
```
## 2. pfile을 spfile과 동기화

```bash
SYS @ orcl > create pfile from spfile;
# open 상태에서도 가능, 덮어쓰기 가능
```

## 3. Database mount 상태로 변경하고 archive log mode 확인

```bash
SYS @ orcl > shutdown immediate
SYS @ orcl > startup mount

SYS @ orcl > archive log list
Database log mode              No Archive Mode
Automatic archival             Disabled
Archive destination            /home/oracle/arch
Oldest online log sequence     4
Current log sequence           7
```
## 4. Database log mode를 archive mode로 변경

```bash
SYS @ orcl > alter database archivelog;

Database altered.

SYS @ orcl > archive log list
Database log mode              Archive Mode
Automatic archival             Enabled
Archive destination            /home/oracle/arch
Oldest online log sequence     4
Next log sequence to archive   7
Current log sequence           7
```

## 5. Database open 상태로 변경하고 system switch logfile 후 archiving 파일 확인

```bash
SYS @ orcl > alter database open;

Database altered.

SYS @ orcl > alter system switch logfile;

System altered.

SYS @ orcl > /

System altered.

SYS @ orcl > archive log list

Automatic archival             Enabled
Archive destination            /home/oracle/arch

Next log sequence to archive   9
```
