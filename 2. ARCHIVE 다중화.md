## 순서
1. archive 추가 경로 생성
2. archive file의 parameter 수정하여 경로 지정
3. 지정된 경로에 archive file 다중화 확인

## 1. archive 추가 경로 생성

```bash
[oracle@ora19c ~]$ mkdir arch2 arch3 arch4
[oracle@ora19c ~]$ ls -l arch*

-rw-r-----. 1 oracle oinstall     66560  4월 13 11:21 ORCLDB_1_8_1229264056.arc

arch2:
합계 0

합계 0

arch4:
합계 0
```

## 2. archive file의 parameter 수정하여 경로 지정
```bash
SYS @ orcl > alter system set log_archive_dest_2='LOCATION=/home/oracle/arch2';

System altered.

SYS @ orcl > alter system set log_archive_dest_3='LOCATION=/home/oracle/arch3';

System altered.

SYS @ orcl > alter system set log_archive_dest_4='LOCATION=/home/oracle/arch4';

System altered.

SYS @ orcl > archive log list
Database log mode              Archive Mode
Automatic archival             Enabled
Archive destination            /home/oracle/arch4
Oldest online log sequence     6
Next log sequence to archive   9
Current log sequence           9

SYS @ orcl > show parameter archive
log_archive_dest_1                   string      LOCATION=/home/oracle/arch
log_archive_dest_2                   string      LOCATION=/home/oracle/arch2
log_archive_dest_3                   string      LOCATION=/home/oracle/arch3
log_archive_dest_4                   string      LOCATION=/home/oracle/arch4
```
